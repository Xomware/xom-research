## Contributing

This repo is internal to Xomware. Add research folders as we explore new tools.

Folder structure convention:
```
{tool-or-topic}/
├── README.md           # Overview — what is it, why does it matter
├── how-it-works.md     # Technical deep-dive
├── use-cases.md        # Practical applications + Xomware-specific notes
└── getting-started.md  # Step-by-step to actually use it
```
